## 🚀 Getting Started

### Prerequisites

* Node.js >= 18
* npm / yarn / pnpm
* Wallet (Phantom, MetaMask, etc.)

### Installation

```bash
git clone https://github.com/yourusername/myth.git
cd myth
npm install
```

### Run Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

## 🔧 Usage

Interact with Myth through:

* Web interface (frontend)
* SDK integration
* Direct smart contract interaction

Example:

```javascript
import { MythSDK } from "myth-sdk";

const myth = new MythSDK();
await myth.connectWallet();
const data = await myth.getState();
```
